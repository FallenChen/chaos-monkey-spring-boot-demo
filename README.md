# chaos-monkey-spring-boot-demo
Demo of Chaos Monkey for Spring Boot 

## How to build
1. git clone https://github.com/MrBW/chaos-monkey-spring-boot-demo.git
2. git checkout chaostoolkit
3. docker-compose build docker-base-image
4. mvn clean package
5. docker-compose up
